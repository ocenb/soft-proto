Задание 1. Настройка маршрутизации

    Установите React Router:

    Добавьте маршрутизацию в приложение с помощью библиотеки React Router.
    Настройте основные маршруты:
    – / — главная страница;
    – /list — страница с списком сущностей;
    – /list/:id — страница с подробной информацией о сущности;
    – /about — страница «О нас».

    Реализуйте переходы между страницами:

    Добавьте навигационное меню с использованием компонента <NavLink>.
    Настройте активное состояние ссылок.

    Создайте компоненты страниц:

    <Home /> — главная страница с приветственным текстом;
    <List /> — страница, отображающая список сущностей;
    <Details /> — страница с деталями сущности, загружаемыми через API;
    <About /> — информация о проекте.

Задание 2. Работа с сетью

    Настройте получение данных для каталога:

    Используйте API, реализованное в рамках модуля бэкенд.
    Загрузите список сущностей на странице /list и отобразите их в виде карточек:
    – название;
    – краткое описание;
    – любая метаинформация.

    Детальная информация о товаре:

    При переходе на /list/:id загрузите подробную информацию о сущностях через API.
    Отобразите:
    название;
    любую метаинформацию;
    полное описание;
    изображение.

    Обработка ошибок:

    Добавьте обработку ошибок при запросах к API:
    если запрос не удался, покажите сообщение об ошибке;
    реализуйте состояние загрузки (спиннер или индикатор).

Задание 3. Управление состоянием

    Создайте глобальное состояние:

    Используйте Context API для хранения списка сущностей или сделайте светлую и тёмную тему в приложении.
    Передавайте данные о товарах между страницами без повторных запросов.

    Реализуйте добавление товара в корзину:

    Добавьте кнопку «Добавить в избранное» на странице сущности.
    Храните список избранных сущностей в списки избранное в Context API.
