Задание 1. Понимание Event Loop

Напишите JavaScript-скрипт, который демонстрирует работу Event Loop.

Используйте:

    console.log() — для синхронного кода;
    setTimeout() — для отложенных задач;
    Promise с методами resolve и then.

Добавьте комментарии, поясняющие порядок выполнения кода.

Результат: .js-файл с кодом и комментариями.

Задание 2. Работа с Promise

Реализуйте цепочку асинхронных операций с использованием Promise:

    Создайте функцию fetchData(url), возвращающую Promise с задержкой в 2 секунды.
    Реализуйте цепочку вызовов:
        Первый запрос к API возвращает список пользователей.
        Второй запрос на основе данных первого выводит информацию о первом пользователе.
    Добавьте обработку ошибок через .catch().

Результат: код, загружающий данные и корректно обрабатывающий ошибки.

Задание 3. Работа с async/await

Перепишите код из Задания 2, используя async/await:

    Используйте конструкцию try…catch для обработки ошибок.
    Реализуйте функцию delay(ms), которая добавляет задержку между запросами.

Результат: код, выполняющий асинхронные операции с async/await.

Задание 4. Прогресс-бар для асинхронного выполнения

Создайте простой прогресс-бар для отображения выполнения асинхронной задачи:

    Создайте HTML-страницу с <div id=“progress-bar”>.
    Напишите скрипт:
        выполнение задачи длится 5 секунд;
        каждую секунду прогресс-бар обновляется, например, ширина заполняется на 20%.
    После завершения задачи выводится сообщение «Готово!».

Результат: HTML-страница с работающим прогресс-баром.
