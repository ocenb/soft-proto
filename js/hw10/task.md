Задание. Создать базовый дизайн системы (UI Kit) для итогового проекта

Разработайте основные элементы интерфейса, которые понадобятся в итоговом проекте:

    кнопки — Button;
    поле ввода — Input;
    выпадающие списки — Dropdown;
    чекбоксы и радиокнопки;
    карточки — Card;
    навигационное меню — Navbar.

Все компоненты должны быть:

    адаптивными;
    стилизованными с использованием методологии БЭМ.

Задача 1. Разработать кнопки

    Реализуйте варианты кнопок:
        основная — Primary;
        второстепенная — Secondary;
        неактивная — Disabled.

    Добавьте поддержку состояний: hover, active, focus.

    Реализуйте 3 размера кнопок: маленький (small), средний (medium), большой (large).

Задача 2. Создать поле ввода

    Реализуйте текстовое поле, поддерживающее состояния:
        обычное — Normal;
        фокус — Focus;
        ошибка — Error — отображение сообщения об ошибке.

    Настройте возможность добавления иконки, например, значка поиска.

Задача 3. Разработать карточки

    Создайте компонент карточки с заголовком, изображением и текстом.

    Настройте модификаторы карточек:
        с изображением / без изображения;
        размеры карточек: маленькая (small), средняя (medium), большая (large).

Задача 4. Реализовать навигационное меню

Реализуйте адаптивное навигационное меню:

    на десктопе — горизонтальное;
    на мобильных устройствах — выпадающее (гамбургер-меню).

Задача 5. Организовать макет с помощью Flexbox и Grid

Реализуйте контейнеры и сетки для выравнивания элементов:

    контейнер с центральным выравниванием контента;
    Grid-сетка для создания галерей или списков карточек.

Задача 6. Организовать SCSS (или CSS) структуру

Разделите стили на:

    переменные: цвета, шрифты, размеры;
    базовые: reset.css, normalize.css;
    стили для компонентов UI Kit.

Задача 7. Создать страницу-документацию

Создайте страницу-документацию для UI Kit, содержащую:

    описание каждого компонента;
    примеры использования: варианты кнопок, полей ввода и прочее.
