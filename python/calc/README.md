# FastAPI Calculator

Это простое FastAPI-приложение, реализующее работу калькулятора.

## Функционал

- Сложение, вычитание, умножение и деление чисел.
- Вычисление сложных математических выражений из строки.

## Установка

1.  Убедитесь, что у вас установлен Python 3.8+.
2.  Клонируйте репозиторий (если применимо) или просто скопируйте файлы.
3.  Установите необходимые зависимости:
    ```bash
    pip install -r requirements.txt
    ```

## Запуск приложения

Для запуска сервера разработки выполните команду:

```bash
uvicorn main:app --reload
```

Сервер будет доступен по адресу `http://127.0.0.1:8000`.

## API

После запуска вы можете получить доступ к интерактивной документации API по адресу [http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs).

### Эндпоинты

- `GET /add?a=<number>&b=<number>` - Сложение.
- `GET /subtract?a=<number>&b=<number>` - Вычитание.
- `GET /multiply?a=<number>&b=<number>` - Умножение.
- `GET /divide?a=<number>&b=<number>` - Деление.
- `GET /calculate?expression=<string>` - Вычисление строкового выражения.
  - **Пример:** `http://127.0.0.1:8000/calculate?expression=(10%2B5)*2`
